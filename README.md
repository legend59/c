### 정렬
```cs
List <string> li = new List<string>();

// #1
li.Sort();  // 정렬 오름차순

// #2 delegate
li.Sort(delegate (string x, stringy) {
    return y.CompareTo(x);  // 내림차순
});

// #3 람다식
li.Sort((string x, string y) => x.CompareTo(y));
```
